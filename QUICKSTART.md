# 快速启动指南

## 🎯 5分钟快速上手

### 第一步：注册管理员账号

1. 打开网站首页
2. 点击右上角"**登录**"按钮
3. 切换到"**注册**"标签
4. 输入用户名和密码（至少6位）
5. 点击"**注册**"

> 💡 **重要**：第一个注册的用户将自动成为管理员！

### 第二步：进入管理后台

注册成功后会自动登录，点击右上角用户名 → "**管理后台**"

### 第三步：编辑个人信息

1. 左侧菜单点击"**个人信息**"
2. 填写您的信息：
   - 姓名：孙可馨
   - 年龄：20
   - 学校：武汉理工大学
   - 专业班级：经济2403
   - 联系方式：13154715568
   - 个人简介：（自由发挥）
3. 上传头像（可选）
4. 点击"**保存更改**"

### 第四步：发布第一篇文章

1. 左侧菜单点击"**文章管理**"
2. 点击"**新建文章**"
3. 填写文章信息：
   ```
   标题：我的第一篇博客
   Slug：my-first-blog
   内容：# 你好，世界！
   
   这是我的第一篇博客文章。
   ```
4. 选择分类（如：生活随笔）
5. 开启"**发布状态**"
6. 点击"**创建文章**"

### 第五步：查看效果

1. 点击左上角"**返回前台**"
2. 在首页查看刚发布的文章
3. 点击文章标题查看详情

## ✅ 完成！

恭喜您已经成功搭建了个人博客！
