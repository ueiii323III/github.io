# 孙可馨个人博客 - 使用说明

## 🎉 欢迎使用

这是一个专为孙可馨打造的个人博客网站，采用温暖的粉色主题设计，功能完善，操作简单。

## 📋 功能概览

### 前台功能
- **首页**：浏览所有已发布的文章，支持分页
- **文章详情**：阅读文章完整内容，支持 Markdown 格式
- **分类浏览**：按分类筛选查看文章
- **关于我**：展示个人信息、简介和友情链接
- **侧边栏**：快速访问个人简介和文章分类

### 后台管理
- **文章管理**：创建、编辑、删除文章，设置发布状态
- **分类管理**：管理文章分类
- **个人信息**：编辑个人资料和头像
- **用户管理**：管理用户角色权限
- **图片上传**：上传文章封面和头像，自动压缩优化

## 🚀 快速开始

### 1. 首次使用 - 注册管理员账号

1. 访问网站首页
2. 点击右上角"登录"按钮
3. 切换到"注册"标签
4. 填写用户名和密码（至少6位）
5. 点击"注册"按钮

**重要提示**：第一个注册的用户将自动成为管理员！

### 2. 登录后台

1. 注册成功后会自动登录
2. 点击右上角用户名下拉菜单
3. 选择"管理后台"进入

### 3. 编辑个人信息

1. 进入管理后台
2. 点击左侧菜单"个人信息"
3. 填写/修改以下信息：
   - 姓名
   - 年龄
   - 学校
   - 专业班级
   - 联系方式
   - 个人简介
   - 头像（可上传图片）
4. 点击"保存更改"

### 4. 创建文章分类

1. 点击左侧菜单"分类管理"
2. 点击"新建分类"按钮
3. 填写分类信息：
   - 分类名称（如：生活随笔）
   - Slug（URL标识，如：life）
   - 描述（可选）
4. 点击"创建"

系统已预设了4个分类：
- 生活随笔
- 学习笔记
- 旅行游记
- 读书感悟

### 5. 发布第一篇文章

1. 点击左侧菜单"文章管理"
2. 点击"新建文章"按钮
3. 填写文章信息：
   - **标题**：文章标题
   - **Slug**：URL标识（建议用英文，如：my-first-post）
   - **摘要**：文章简介（可选）
   - **内容**：文章正文，支持 Markdown 格式
   - **分类**：选择文章分类
   - **封面图片**：上传封面图（可选）
   - **发布状态**：开启表示立即发布，关闭表示保存为草稿
4. 点击"创建文章"

### 6. 编辑文章

1. 在"文章管理"页面找到要编辑的文章
2. 点击文章右侧的"编辑"图标
3. 修改文章内容
4. 点击"更新文章"

### 7. 删除文章

1. 在"文章管理"页面找到要删除的文章
2. 点击文章右侧的"删除"图标
3. 在确认对话框中点击"删除"

## 📝 Markdown 使用指南

文章内容支持 Markdown 格式，以下是常用语法：

### 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 文本样式
```markdown
**粗体文本**
*斜体文本*
~~删除线~~
```

### 列表
```markdown
- 无序列表项1
- 无序列表项2

1. 有序列表项1
2. 有序列表项2
```

### 链接和图片
```markdown
[链接文字](https://example.com)
![图片描述](图片URL)
```

### 引用
```markdown
> 这是一段引用文字
```

### 代码
```markdown
`行内代码`

​```
代码块
​```
```

## 🖼️ 图片上传说明

### 支持的格式
- JPEG
- PNG
- GIF
- WEBP
- AVIF

### 文件大小限制
- 最大 1MB
- 超过 1MB 会自动压缩为 WEBP 格式
- 自动调整分辨率至 1080p

### 文件名要求
- 只能包含英文字母、数字、下划线和连字符
- 示例：`my-photo.jpg`、`avatar_2024.png`

## 👥 用户管理

### 角色说明
- **管理员**：拥有所有权限，可以管理文章、分类、个人信息和用户
- **普通用户**：只能查看前台内容

### 修改用户角色
1. 进入"用户管理"页面
2. 找到要修改的用户
3. 在"操作"列的下拉菜单中选择新角色
4. 系统会自动保存

## 🎨 主题特色

- **主色调**：温暖的粉色系 (#FFB6C1)
- **设计风格**：圆润卡片、柔和阴影、简约图标
- **布局**：左侧主内容区 + 右侧个人简介和分类导航
- **响应式**：完美适配桌面和移动设备

## 💡 使用技巧

1. **草稿功能**：创建文章时关闭"发布状态"，可以保存为草稿，前台不会显示
2. **文章摘要**：建议填写摘要，会在首页文章列表中显示
3. **封面图片**：上传封面图可以让文章更吸引人
4. **分类管理**：合理使用分类可以让文章更有条理
5. **Markdown 预览**：建议使用 Markdown 编辑器预览效果后再发布

## 🔒 安全提示

1. 请妥善保管管理员账号密码
2. 不要将管理员权限随意授予他人
3. 定期备份重要文章内容
4. 上传图片前请确认图片内容合适

## 📞 技术支持

2. 浏览器是否为最新版本
3. 是否已正确登录
