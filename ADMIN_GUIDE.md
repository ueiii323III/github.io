# 管理后台使用指南

## 概述

为您的个人博客网站添加了完整的后端管理功能，包括作品管理、图片管理、网站设置等功能。

## 功能特性

### 🔐 认证系统
- 管理员登录认证
- 路由权限保护
- 登录状态持久化

### 📊 仪表盘
- 网站数据统计
- 快捷操作入口
- 最新作品展示
- 系统信息概览

### 🎨 作品管理
- 作品的增删改查
- 分类管理（摄影、绘画、数字艺术、城市印象）
- 批量操作（发布、隐藏、删除）
- 搜索和筛选功能
- 标签管理
- 发布状态控制

### 📸 图片管理
- 校园图片上传和管理
- 拖拽排序功能
- 图片预览
- 批量管理

### ⚙️ 网站设置
- 基本信息配置
- 头像和封面上传
- 社交链接管理
- SEO设置

## 访问方式

### 前台网站
- 首页：`http://localhost:3001/`
- 作品展示：`http://localhost:3001/portfolio`
- 关于页面：`http://localhost:3001/about`

### 管理后台
- 登录页面：`http://localhost:3001/admin/login`
- 仪表盘：`http://localhost:3001/admin/dashboard`
- 作品管理：`http://localhost:3001/admin/works`
- 校园图片：`http://localhost:3001/admin/campus`
- 网站设置：`http://localhost:3001/admin/settings`

## 默认账号

- **用户名**：`admin`
- **密码**：`admin123`

> ⚠️ **安全提醒**：在生产环境中，请务必修改默认密码！

## 使用步骤

### 1. 启动开发服务器
```bash
npm run dev
```

### 2. 访问管理后台
1. 打开浏览器访问 `http://localhost:3001/admin/login`
2. 使用默认账号登录
3. 登录成功后会跳转到仪表盘

### 3. 管理作品
1. 在左侧菜单点击"作品管理"
2. 可以查看、添加、编辑、删除作品
3. 支持按分类、状态筛选
4. 支持批量操作

### 4. 管理校园图片
1. 在左侧菜单点击"校园图片"
2. 可以上传新的校园图片
3. 支持拖拽排序
4. 可以编辑图片信息

### 5. 配置网站
1. 在左侧菜单点击"网站设置"
2. 修改基本信息、社交链接、SEO设置
3. 上传头像和封面图片
4. 点击"保存设置"

## 数据存储

目前使用 **localStorage** 进行数据持久化：
- 作品数据存储在 `localStorage.works_data`
- 网站设置存储在 `localStorage.site_settings`
- 校园图片存储在 `localStorage.campus_images`
- 用户认证存储在 `localStorage.admin_user`

> 💡 **提示**：数据存储在浏览器本地，清除浏览器数据会丢失所有管理内容。

## 常见问题

### Q: 如何修改管理员密码？
A: 目前密码是硬编码的，可以修改 `stores/auth.ts` 中的登录逻辑。

### Q: 图片上传不成功？
A: 当前使用模拟上传，需要配置真实的文件上传API。

### Q: 数据丢失了怎么办？
A: 由于使用localStorage，清除浏览器数据会丢失。建议定期导出数据备份。

### Q: 如何自定义分类？
A: 可以修改 `types/admin.ts` 中的 `WorkCategory` 类型定义。
