# 孙可馨的个人空间

> 个性化个人展示平台 - 专注绘画与摄影作品展示

## 项目简介

为武汉理工大学大二学生孙可馨打造的个性化个人展示平台，重点突出绘画与摄影作品展示功能，同时整合校园生活记录与个人成长轨迹。

## 技术栈

- **前端框架**: Vue.js 3 + TypeScript
- **UI组件库**: TDesign Vue Next
- **构建工具**: Vite
- **状态管理**: Pinia
- **路由**: Vue Router 4
- **样式**: SCSS + CSS Variables
- **3D图形**: Three.js (计划中)
- **地图**: Leaflet.js (计划中)
- **动画**: GSAP (计划中)

## 安装运行

```bash
# 克隆项目
git clone [repository-url]
cd 个人博客

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 设计理念

采用**现代艺术极简主义**设计风格，融合玻璃态元素与柔和渐变，打造专业而优雅的视觉体验。

- **主色调**: 深蓝色 (#1e3a5f) 与浅灰白 (#f8f9fa)
- **点缀色**: 金色 (#d4af37) 
- **字体**: 思源黑体 + Inter
- **效果**: 毛玻璃、柔和阴影、平滑过渡
